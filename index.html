Descrição

O projeto é um site de conscientização sobre a COVID-19, que apresenta dados atualizados e gráficos interativos para informar o público sobre casos, mortes e recuperações. O objetivo é educar e promover medidas de prevenção, combatendo a desinformação.

Objetivo do projeto

O objetivo deste projeto é criar uma plataforma informativa que conscientize e eduque a população sobre a COVID-19, fornecendo dados precisos e atualizados, promovendo a importância das medidas de prevenção e combatendo a desinformação sobre a pandemia.

Especificação das Tecnologias e Ferramentas Utilizadas

Especificação das Tecnologias e Ferramentas Utilizadas: HTML e CSS foram usados para estruturar e estilizar o site, enquanto JavaScript proporcionou interatividade. Google Charts criou gráficos interativos e uma API de Dados de COVID-19 forneceu informações atualizadas. E editores de código como Visual Studio Code foram utilizados para a codificação do projeto.
