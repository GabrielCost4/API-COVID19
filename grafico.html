Descrição do Tema

Este projeto visa mostrar os casos confirmados de COVID-19 em diferentes países, além de apresentar gráficos e dados referentes ao Brasil. O objetivo é apresentar informações de forma visual e interativa.

Objetivo

O objetivo é fornecer uma visão clara e compreensível sobre os dados de COVID-19, utilizando tecnologias modernas e acessíveis, como Google Charts e API de dados públicos.
